# Demo scenario

1. Загрузить все файлы из `demo_data/`.
2. Задать вопрос: `Какие параметры указаны для клапана DN50?`
3. Проверить, что ответ содержит DN50, PN16, 12Х18Н10Т и ГОСТ 33259.
4. Задать вопрос: `Какие артикулы относятся к насосу NPK-200?`
5. Проверить артикулы ART-NPK-200-BODY, ART-NPK-200-IMP и SEAL-45X60.
6. Задать вопрос: `Где в документации не хватает данных по коррозионной стойкости 7075-T6?`
7. Проверить, что система возвращает gap по коррозионной стойкости.
